job Type
    -Front-end
        A front-end developer creates the parts of a website or app that users see and interact with. 
        They use HTML, CSS, JavaScript, and tools like React, Angular, and Vue.js to make websites look good and work well on all devices and browsers.
        
    -Back-end 
         A back-end developer builds and manages the parts of a website or app that users don’t see.
          They work with databases, servers, and application logic to process data and ensure everything runs smoothly.
           Common languages and frameworks include Node.js, Python (Django, Flask), Java (Spring), and PHP. 

    -DevOps
        pipeline(operation manegaement)
           DevOps is a practice that combines development and operations to deliver software faster and more reliably.
            It focuses on automation, continuous integration (CI), continuous delivery (CD), and monitoring using tools like Docker, Kubernetes, Jenkins, and Git.

    -Fullstack
        A full-stack developer works on both the front-end (user interface) and back-end (server, database) of web applications. They use technologies like HTML, CSS, JavaScript for the front end and Node.js, Python, Java for the back end.

    -Tester
        A tester ensures software works correctly by finding bugs and performance issues.

          -  Manual Testing: Testing software by hand to find errors.
            -A Business Analyst (BA) gathers requirements and helps ensure the product meets business needs.
          -  Automation Testing: Using tools like Selenium to run tests automatically.

    -Cloud engineering 
        It is the design, development, and management of cloud systems using platforms like AWS, Azure, and Google Cloud for secure and scalable solutions
